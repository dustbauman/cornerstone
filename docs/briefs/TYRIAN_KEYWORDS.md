## Programmatic SEO — URL patterns

These are not single keywords. They are URL templates that auto-generate hundreds of indexable pages from directory data. Build in this order.

```
/directory/[state]                     → e.g. /directory/florida
/directory/[trade]                     → e.g. /directory/contractors
/directory/[state]/[trade]             → e.g. /directory/florida/plumbers   ← BUILD FIRST
/directory/[city]/[trade]              → e.g. /directory/sarasota/attorneys  (scale with member density)
/lodge/[lodge-slug]                    → e.g. /lodge/sarasota-lodge-147
```

**Rules for programmatic pages:**
- Never build a city-level page with zero listings — thin pages hurt domain authority
- Every page title should follow: `[Trade] in [Location] | Lodge-Verified Professionals | TYRIAN`
- H1 pattern: `Lodge-Verified [Trade] in [City/State]`
- Meta description pattern: `Find trusted, lodge-verified [trade] in [location] through TYRIAN, the Masonic professional referral network. Every professional is vouched for by their lodge.`
